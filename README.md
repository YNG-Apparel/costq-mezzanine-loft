# Cost-Q Mezzanine Loft

A responsive property showcase website designed to present accommodation information, amenities, location details, and media galleries for a mezzanine loft rental.

## Live Demo

https://cost-q.netlify.app/

## Features

- Responsive design for desktop and mobile devices
- Property information and accommodation details
- Amenities showcase
- Photo gallery
- Location information
- Interactive user interface
- Modern visual design

## Technologies Used

- HTML5
- CSS3
- JavaScript
